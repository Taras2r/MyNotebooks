Mathematical Model of PI - controller

The model is based on discrete form of differential equations

The inspiration has been taken from: Industrial IT and Automation Channel

Very useful resource to write formulas with Markdown: Mathematics in Markdown

The mathematical model of the system to be controlled by PI - controller is the following differential equation: $$\frac{\partial y}{\partial t} = ay + bu \tag{1}$$

The mathematical model of the PI - controller is represented as the following differential equation: $$u(t) = K_{p}e + \frac{K_{p}}{T_{i}} \int_{0}^{t} e \, d\tau \tag{2}$$
