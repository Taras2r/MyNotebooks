NbConverter documentation

Learned commands those are usefull to convert a Jupyter notebook to html:

  • to remove as sell input so as sell output:
jupyter nbconvert --to html --TagRemovePreprocessor.enabled=True --TagRemovePreprocessor.remove_cell_tags hide_cell Controls.ipynb
  • to leave cell output but remove the input:
jupyter nbconvert --to html --TagRemovePreprocessor.enabled=True --TagRemovePreprocessor.remove_input_tags hide_cell Controls.ipynb
